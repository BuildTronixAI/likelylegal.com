# LikelyLegal? — Ideas that sound legal yet dumb.

A daily question-answer site about dumb ideas that might actually be legal. Short, confident, slightly irreverent answers for contractors, small business owners, and entrepreneurs.

## Concept

**"Ideas that sound legal yet dumb."** with a question mark as the brand.

- **Tagline:** "We're not lawyers. But we're probably right."
- **Secondary:** "You probably shouldn't. But let's find out."
- **Audience:** Contractors, small business owners, entrepreneurs
- **Format:** One daily question, short answer, confidence rating (🟢 / 🟡 / 🔴)

## Content Categories

1. Construction/Contractor law
2. Small business/Tax
3. Employment law
4. Real estate
5. Tax/Financial
6. General business absurdity
7. Florida-specific (primary market)

## Tech Stack

- **Static HTML/CSS/JavaScript** - No build process required
- **Deployment:** Vercel
- **Domain:** likelylegal.com (GoDaddy DNS)

## Design

- Dark theme (#0a0a0a background)
- BrocastStudios family aesthetic
- Question mark (?) is prominent in hero
- Mobile-first, responsive
- Dark cards with accent orange (#ff6b35) highlights

## Running Locally

```bash
npm run dev
# Opens on http://localhost:3000
```

## Deployment

Deployed to Vercel as a static site. DNS configured via GoDaddy API to point to Vercel.
